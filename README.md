# NFT Marketplace

A basic NFT Marketplace smart contract using Solidity, Hardhat, and OpenZeppelin.

## Features
- Mint NFTs
- List NFTs for sale
- Buy NFTs with ETH

## Quickstart
```bash
npm install
npx hardhat test
```

## Deployment
```bash
npx hardhat run scripts/deploy.js --network hardhat
```
